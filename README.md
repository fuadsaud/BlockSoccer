BlockSoccer
===========

Soccer game built directly on top of OpenGL and GLUT for learning purposes.

![FIFA? Bulshit!](screenshot.png)

```w```, ```a```, ```s```, ```d``` - move the player  
```space``` - kick the ball

that's it.
